**Final Project Proposal**

RaagaSearch

A Digital Database of North Indian Ragas: Swaras, Scale Degrees, and Cross-System Equivalencies

A searchable web-based database designed to map North Indian ragas through swaras, scale degrees, and their equivalents in both Western and Carnatic musical systems.

Resources:
https://www.tanarang.com
https://raagabase.com
https://www.swarganga.org
These resources will be used to study the way the databases are designed.

This project proposes the development of a searchable digital database that organizes North Indian ragas through multiple theoretical frameworks, including swaras (Indian solfege), scale degrees, Western scale equivalencies, and Carnatic raga parallels. A key feature of this project will be a search function that allows users to easily navigate and identify ragas based on different parameters such as scale structure or note combinations. The goal is to create an accessible and practical tool for musicians, composers, and producers working across musical systems.

No, this project does not overlap with any work from another class.

GOOD Outcome: A functional, searchable database that includes a set of ragas with their corresponding swaras and scale degrees.

BETTER Outcome: A searchable database that includes swaras, scale degrees, and their Western and Carnatic equivalencies for each raga.

BEST Outcome: A searchable database that includes swaras, scale degrees, equivalencies, and additional contextual information such as raga mood, emotional character, and performance considerations.
